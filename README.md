## Jupyter notebook 安装 javascript kernel

### 安装 python、pip

**pip更换豆瓣源**

以`window`为例子，新建`%APPDTA%/pip/pip.ini`文件，粘贴一下内容

```
[global]
time-out=60
index-url=https://pypi.douban.com/simple/
[install]
trust-host=pypi.douban.com
```

### 安装ijavascript

[ijavascript](https://github.com/n-riesco/ijavascript) github地址

```shell
npm install -g ijavascript

jp-babel-install

```
